Instruction for the exercise:
Develop an abstraction where each instance stores a sequence of int values. Allow clients to create an IntList object that stores the sequence of int values given in an array, to retrieve an array containing the sequence of int values currently stored by an IntList object, to add a value to the end of the sequence of int values stored by an IntList object, and to remove the last value in the sequence of int values stored by an IntList object.

First define the syntax and semantics of the API, and a complete test suite, and then develop two implementations: one which stores the sequence of int values in an array, and other one that stores it in a linked list of nodes. (See the JLearner examples to refresh the concept of a linked list of nodes.)
